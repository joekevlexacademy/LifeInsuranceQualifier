Set up in 60 seconds

Connect your GoHighLevel sub-account and configure everything automatically.

  • ✓ Creates all 15 qualification contact fields
  • ✓ Adds a sidebar menu link for quick access
  • ✓ No manual steps required
You'll need: GoHighLevel admin access and a Private Integration key for the sub-account.

Enter Location ID

Paste your GHL sub-account Location ID to begin setup.

Where to find it:
In GHL, open the sub-account. The ID is in the URL:
app.gohighlevel.com/v2/location/YOUR_ID/...

Choose sub-account

Select the sub-account to configure.

Integration keys

Connect to Life Insurance Qualifier.

Sub-account key (required)
Switch into the sub-account in GHL → Settings → Private Integrations → New Integration.
Scopes: Contacts, Custom Fields, Notes, Locations — or select all.
Agency key (optional — enter once per agency)
Lets the app automatically create and update the sidebar menu link for all sub-accounts.
From the agency view → Settings → Private Integrations → New Integration.
Scope required: Edit Custom Menu Links (custom-menu-link.write)
Leave blank if already entered for this agency.
